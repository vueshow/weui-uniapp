# 主题样式

组件库样式来自 WeUI，并通过 CSS 变量保持颜色、背景和状态层一致。

## 样式入口

```scss
@import 'weui-uniapp/styles/index.scss';
```

## CSS 变量

常用变量包括：

| 变量 | 说明 |
| --- | --- |
| `--weui-BRAND` | 品牌主色 |
| `--weui-BG-0` | 页面背景 |
| `--weui-BG-2` | 卡片与弹层背景 |
| `--weui-FG-0` | 主要文字 |
| `--weui-FG-1` | 次要文字 |

后续可以在这里补充暗色模式、适老化模式和组件级变量说明。
